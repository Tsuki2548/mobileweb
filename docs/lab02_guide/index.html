แนะนำสถานที่ใน มข.

บึงสีฐาน

บึงสีฐานเป็นแหล่งเก็บน้ำธรรมชาติขนาดใหญ่ของมหาวิทยาลัยขอนแก่น ที่มีความสำคัญต่อระบบนิเวศและการศึกษาด้านสิ่งแวดล้อม

คอมเพล็กซ์ (ศูนย์อาหารและบริการ)

ศูนย์อาหารและบริการที่ให้ประโยชน์แก่นักศึกษา อาจารย์ และบุคลากรมหาวิทยาลัย พร้อมด้วยห้องน้ำและสถานที่พักผ่อน

หอสมุดกลาง

หอสมุดกลางเป็นศูนย์กลางการค้นหาข้อมูลและการเรียนรู้ของมหาวิทยาลัย มีทรัพยากรสารสนเทศและสถานที่ศึกษาที่สมบูรณ์

โรงพยาบาลศรีนครินทร์

โรงพยาบาลศรีนครินทร์ให้บริการสุขภาพแก่นักศึกษา อาจารย์ บุคลากร และประชาชนในพื้นที่รอบมหาวิทยาลัยขอนแก่น
